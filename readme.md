FTP file download and reader.

### How to use it

1. Edit file names  
 `$localFile = 'exercise.txt';`  
 `$remoteFile = 'exercise.txt';`
 
2. Edit connection parameters  
 `$ftp->connect('test.nl');`  
 `$ftp->login('user', 'passwd');`

3. run   (command line)
 `php src/exercise.php`
    
    
### Requirements
PHP >= 7.1
